# Form Validation System

A login form validation project that validates user input before form submission.

## Features
- Email validation
- Password validation
- Error messages
- Success notification
- Real-time visual feedback

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Validation Rules
- Email must follow a valid format
- Password must contain at least 6 characters

## Learning Outcomes
- Form Validation
- Regular Expressions
- Event Handling
- User Experience Design
